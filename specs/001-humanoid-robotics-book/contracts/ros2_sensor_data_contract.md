# Contract: ROS 2 Sensor Data Publishing

## Purpose
To publish simulated sensor data (e.g., LiDAR, IMU) over ROS 2 topics.

## ROS 2 Topics
- `/robot/lidar/points`
- `/robot/imu/data`

## Message Types
- `/robot/lidar/points`: `sensor_msgs/PointCloud2`
- `/robot/imu/data`: `sensor_msgs/Imu`

## Inputs
- None (sensor data is generated by the simulation or physical sensor).

## Outputs
- `/robot/lidar/points`: Point cloud data from the LiDAR sensor.
- `/robot/imu/data`: Inertial measurement unit data (acceleration, angular velocity, orientation).

## Error Taxonomy
- No explicit error responses for topic publishing; data loss or corruption is handled by ROS 2 messaging robustness.
